# External Data Notes

통합 전에 최신 약관과 문서를 다시 확인한다.

- yfinance project: https://github.com/ranaroussi/yfinance
- yfinance docs: https://ranaroussi.github.io/yfinance/
- Yahoo API terms: https://legal.yahoo.com/us/en/yahoo/terms/product-atos/apiforydn/index.html
- Alpha Vantage docs: https://www.alphavantage.co/documentation/
- Alpha Vantage MCP: https://mcp.alphavantage.co/
- Stooq historical data: https://stooq.com/db/h/
- FinanceDataReader: https://github.com/FinanceData/FinanceDataReader
- KRX data: https://data.krx.co.kr/

yfinance는 Yahoo가 승인한 공식 SDK가 아니며 연구·교육 목적의 오픈소스 도구로 설명된다. 이 문서는 데이터 라이선스를 대신하지 않는다.
