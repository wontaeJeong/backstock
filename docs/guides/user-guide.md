# User Guide

1. Market Data에서 종목, 기간, provider를 선택한다.
2. 데이터를 수집하거나 CSV/Parquet를 업로드한다.
3. 품질 검사와 경고를 확인하고 snapshot을 만든다.
4. Strategy에서 템플릿 또는 조건을 작성한다.
5. immutable 전략 버전을 저장한다.
6. Backtest Setup에서 자본, 비용, 슬리피지, 체결 정책을 설정한다.
7. 실행 후 성과, 위험, 주문, 체결, 포지션을 확인한다.
8. 다른 실행과 비교한다.

높은 수익률만 보지 말고 최대 낙폭, turnover, 비용, 거래 수, provider, adjusted/raw, 품질 경고를 함께 확인한다. Paper order는 실제 주문이 아니다.
